LED Matrix Controller

The LED Matrix Display via Android App is a wireless system where users control an LED matrix using a custom Android app through Bluetooth (HC-05). The app lets users send text, adjust scroll speed, and change brightness in real time. An Arduino with MAX7219 handles smooth text display and animations. A dedicated website is also created to host the app and provide documentation. This project showcases seamless communication between hardware and software. Technologies: Arduino IDE, Android (Java), Bluetooth HC-05, MAX7219 LED Matrix, Web Hosting.
