LED Matrix Controller

The LED Matrix Display via Android App project is a wireless display system that allows users to control an LED matrix through a custom-built Android application using Bluetooth. The app enables users to send custom text messages, adjust scroll speed, and control the brightness of the display in real time. The Arduino, programmed using the MAX7219 LED driver, manages smooth text rendering and animation on the LED matrix. Additionally, a dedicated website was developed to host the Android app and provide documentation for user guidance and support. This project effectively integrates hardware and software, showcasing seamless communication between the Android app and the Arduino-based LED display. Technologies Used: Arduino IDE, Android (Java), Bluetooth (HC-05), MAX7219 LED Matrix, Web Hosting.
